Hand Ranks

The table below shows all the combinations of cards that increase a hands value in poker. If an opponent has a combination that is higher in the list than your hand, then he/she will win. In texas hold'em, there are seven cards that can be matched, but only the top 5 cards are counted.

Hand Name Example Description Odds
Royal Flush Ace through 10 with cards of the same suit 649,739 : 1
Straight Flush Five consecutive cards of the same suit 72,192 : 1
Four of a Kind Four cards with the same value 4,164 : 1
Full House Three cards with the same value, and two cards with the same value 693 : 1
Flush Five cards of the same suit 508 : 1
Straight Five consecutive cards 254 : 1
Three of a Kind Three cards with the same value 46.3 : 1
Two Pair Two cards with the same value and two cards with the same value 20.0 : 1
Single pair Two cards with the same value 1.36 : 1
High Card Highest card wins, if tied, second highest card, if tied third, if five are the same, split 0.995 : 1
